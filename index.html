Using GitHub

How to Upload Files to Your Git Repository
1. Create a git repository using GitHub.
2. In your project file, use "git init -b main" to initialize the local directory as a Git repository
3. Use "git add ." to add files to your Git local repository
4. Use "git commit -m 'message'" to commit the files that you have staged in your local repository
5. Add the link to your repository on GitHub using "git remote add origin 'url'"
6. Push the local repository to the GitHub repository by "git push origin main"
Reset the link to the remote repository
Use the command "git remote set-url origin 'new url'"
